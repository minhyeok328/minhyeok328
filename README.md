## 📌 Projects
- **1st_project — TCO Insight (차량 운영/관리 비용 계산 시스템)**  
  차량 연비/가격/유가 API와 DB를 연동해 차량별 월간·연간 운영비(TCO)를 계산하고, Streamlit UI로 비교 가능하게 만든 프로젝트입니다.  
  **기여 포인트:** API 데이터 파싱, CSV/DB 적재 구조 구성, 유지비 계산 로직 구현 및 시각화 화면 연동.

- **2nd_project — 신용카드 고객 이탈 예측 시스템**  
  Kaggle 신용카드 데이터를 기반으로 ETL 파이프라인, FastAPI, Streamlit, MLflow, Docker 환경을 구성하고 XGBoost/LightGBM 등으로 이탈 확률을 예측한 MLOps형 프로젝트입니다.  
  **기여 포인트:** 데이터 파이프라인 설계/구현, XGBoost 기반 모델링, Unknown 데이터 예측 실험, 실험 결과 해석 및 전략 연결.

- **3rd_project — PICKLE 맛집 추천 챗봇**  
  신대방삼거리 식당 데이터를 수집·가공해 SQLite + 임베딩 검색을 구축하고, LangGraph RAG 파이프라인으로 자연어 질의에 근거 기반 맛집 추천을 제공한 LLM 서비스 프로젝트입니다.  
  **기여 포인트:** LLM 응답 파이프라인 설계, 임베딩 기반 검색 흐름 연결, Streamlit UI 연동, 평가 리포트 기반 품질 개선.
